# Next Chat Android

Next Chat is a secure enterprise messaging application for internal corporate communication, based on the Matrix protocol.

## About

Next Chat Android is built on [Element Android](https://github.com/element-hq/element-android), an open source Matrix Client. This is a customized version specifically designed for enterprise internal communication.

### Key Features

-   End-to-end encrypted messaging
-   Voice and video calls
-   File sharing and collaboration
-   Enterprise-grade security
-   Self-hosted Matrix server support
